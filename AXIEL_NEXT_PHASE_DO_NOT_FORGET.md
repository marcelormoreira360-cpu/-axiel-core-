# AXIEL Core — Próxima fase obrigatória

Depois da Execução 03, seguir para o fluxo inteligente:

## Form → Session → Insight → Snapshot

Objetivo: conectar os dados já criados.

1. Form preenchido pelo paciente
2. Respostas aparecem dentro da Session
3. AI organiza os dados em um Insight
4. Profissional valida o Insight
5. Snapshot da próxima Session mostra:
   - Latest Insight
   - Last Session
   - Key notes
   - Next Step
   - Product Support, quando existir

Esta fase transforma o AXIEL de um software com módulos separados em uma plataforma inteligente de jornada do paciente.
